Tic-Tac-Toe in C++

A simple console-based Tic-Tac-Toe game written in C++. It allows two players to play against each other in the terminal.

How to Run

Compile the program:
g++ main.cpp -o tictactoe

Run it:
./tictactoe

Features

Two-player gameplay

Command-line interface

Simple and beginner-friendly C++ code
